HTML Tags

The elements on a webpage that specify how your web browser should format and present your page are known as HTML tags. Common HTML tags include:

<div>
Defines a division or section in an HTML document. It is a container for content that allows styling with CSS.
Example:
This is a division.
<p>
Defines a paragraph. Used to display text content in a block.
Example:

A paragraph is a group of sentences that fleshes out a single idea.
In order for a paragraph to be effective, it must begin with a topic sentence,
have sentences that support the main idea of that paragraph, and maintain a consistent flow.

<a>
Defines a hyperlink. Used to link to another web page or resource.
Example: Open youtube
<img>
Defines an image. Used to embed images in a web page.
Example:
<ul>
Defines an unordered list. Contains list items that are typically displayed with bullet points.
Example:
  • Orange
  • Mango
  • Kiwi
<ol>
Defines an ordered list. Contains list items that are typically displayed with numbers or letters.
Example:
  1. Grade 1
  2. Grade 2
  3. Grade 3
<li>
Defines a list item. Used within <ul> or <ol> to represent individual items in a list.
Example:
  • 1 kg Rice
  • 2 kg flour
  • 1/2 kg sugar
<table>
Defines a table. Used to organize data into rows and columns.
Example:
Data 1 Data 2
Data 3 Data 4
<tr>
Defines a table row. Used within <table> to represent a row of table data.
Example:
Data 1 Data 2
Data 3 Data 4
<td>
Defines a table cell. Used within <tr> to represent individual cells in a table row.
Example:
Data 1 Data 2
Data 3 Data 4

CSS Properties

An HTML element's style or behavior is typically assigned by a CSS attribute. Common CSS properties include:

color
Sets the color of text.
Example: p { color: #f0f0f0; }
font-size
Sets the size of the font.
Example: p { font-size: 16px; }
background-color
Sets the background color of an element.
Example: div { background-color: #f0f0f0; }
margin
Sets the margin around an element.
Example: div { margin: 10px; }
padding
Sets the padding inside an element.
Example: div { padding: 10px; }
border
Sets the border of an element.
Example: div { border: 1px solid #ccc; }
text-align
Sets the alignment of text.
Example: p { text-align: center; }
display
Specifies how an element is displayed.
Example: div { display: inline-block; }
